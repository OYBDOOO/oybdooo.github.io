# 一个悲惨的经历

班$rk$后$5$，年$rk100+$，有史以来最惨的一次。



## day1

------------


语文（$107pts$）：选择错四个注定我凉了。作文还$-20pts$

总结：平常不好好做题考试还想考好？


------------

物理（$84pts$）：选择还是扣四个，都是不该错的，大题秀积分$-2pts$

总结：觉得自己做题很难放下心态，总想着做出来题，但不抓分，应该谨慎细心为好。



------------
化学（$69pts$）：啥都扣一堆。

总结：我化学从年$rk1$嘚瑟到了$rk300+$这是最近没做题的缘故，或者做题太粗糙，不求甚解。


------------
## day2



------------
数学（$113pts$）:填空对一个，大题错一堆。

总结：考的好难，做完选择心态炸裂，在去搞前几道大题心态爆炸，整填空的时候就没啥感觉了，总是坠着心，后来还没答完，深受上回巨算题影响，没往简单方法上想。下次一定要把握好这个度，不要放弃思考地硬算，也不要在思考中执迷不悟。



------------
英语（$118pts$）：听力错的多，七选五错俩，完型错六个。

总结：之前完型落的坑终于体现出来了，我卑微的词汇量与巨慢的答题速度，看来坚持不背$3e3+5e2$词的我还是得妥协（$bushi$



------------
生物（$76pts$）：前面扣了$16pts$，后面还可以。

总结：生物与上次比有了进步，但是还没到达高水平，下次努力。



------------
## 总体总结

### 这次好惨的原因，我觉得并不像$Adu$所说是因为我准备竞赛，其实是因为我上个月水作业所带来的隐患所致，即使下一个阶段也是繁忙的竞赛季，但是我觉得我从今天开始细细理解作业上的每个题目不会像这次这样失分。但是在这种状态的话我可能就是$rk38$了。
